Simle script that creates *.txt structure for one year like so:

2020/01_january:
	01.txt
		date 01.01.2020
		date 02.01.2020
		date 03.01.2020
		date 04.01.2020
		date 05.01.2020
	02.txt
		date 06.01.2020
		date 07.01.2020
		date 08.01.2020
		...
	...
2020/02_february
	01.txt
		date 03.02.2020
		...
	02.txt
...

and so on.

It is used to create digital diary, where you can list your every day tasks. Works for every year.
